# Doink
A 2D game played 2 players on the same keyboard.

*Controls*
Player 1: WASD + space

Player 2: ARROWKEYS + .

### Run
If Lua and Löve2D is installed on your system you can run this game like all
other games.

### Build
Create zip-files for Mac and Windows ready to be distributed by running 
$ ./build.sh
on Mac or Linux.
